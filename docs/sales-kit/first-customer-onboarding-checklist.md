# First Customer Onboarding Checklist

## Before Setup

- Confirm garage name and contact person.
- Confirm selected plan.
- Confirm founder offer eligibility.
- Confirm payment terms.
- Confirm WhatsApp/WABA approach.
- Confirm first pilot workflow.

## Data Needed

- Staff users and roles.
- Lead/client list, if available.
- Services offered.
- Booking slots.
- Current WhatsApp number.
- First test lead/customer.

## Technical Setup

- Create company/tenant.
- Create admin and manager users.
- Configure lead source.
- Configure WhatsApp only after approval.
- Confirm queue worker plan.
- Verify first demo flow.

## Training

- Owner/Admin dashboard.
- Manager dashboard.
- Inbox.
- Lead to opportunity.
- Opportunity to booking.
- Booking to job.
- Job to invoice.
- Retention angle.

## Go-Live Checks

- No real messages sent before approval.
- Templates approved.
- Test inbound complete.
- Test outbound approved.
- Mobile view approved.
- Support contact agreed.
